# to_ergo_mas
Αποθετήριο του Δημοτικού Σχολείου Φιλώτα για στην συμμετοχή στον Διαγωνισμό  Ανοιχτών Τεχνολογιών
Οι μαθητές/-τριες θα συμμετέχουν στον σχεδιασμό και την υλοποίηση της μακέτας χρησιμοποιώντας Arduino, micro:bit, αισθητήρες, LED και απλό προγραμματισμό, προσαρμοσμένο στις δυνατότητές τους, αναπτύσσοντας δεξιότητες STEM.
Στόχοι:
οι μαθητές/-τριες
να ευαισθητοποιηθούν σε θέματα οδικής ασφάλειας και προσβασιμότητας στον αστικό χώρο.
να κατανοήσουν τις δυσκολίες που αντιμετωπίζουν τα άτομα με αναπηρία (ΑμεΑ) κατά τη μετακίνησή τους.
να αναγνωρίζουν και να διαχωρίζουν τη σωστή από τη λανθασμένη κυκλοφοριακή συμπεριφορά.
να αξιοποιήσουν ανοιχτές τεχνολογίες για τη δημιουργία ενός λειτουργικού εκπαιδευτικού συστήματος.
να αναπτύξουν δεξιότητες συνεργασίας, δημιουργικότητας και επίλυσης προβλημάτων.
να καλλιεργήσουν ενσυναίσθηση και υπεύθυνη κοινωνική συμπεριφορά.

θα χρειαστούμε:
3D εκτυπωτής, lego, Arduino, micro:bit, LED (κόκκινο – πράσινο), Αισθητήρες απόστασης, Αισθητήρες χρώματος,  Μακετόχαρτο, Καλώδια.
